## Executando
entre na pasta do simulador:

```bash
cd src
```

execute o pacote:

```bash
python -m simcampus
```

é possivel executar passando parametros, exemplo que alterar os dias da simulação para 15:
```bash
python -m simcampus -d 15
```

Para ver todos os parametros possiveis:
```bash
python -m simcampus --help
```
